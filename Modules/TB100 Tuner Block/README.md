# TB100 Tuner Block

**These parts needs to be strong!** The tuner block and nut hold the tension from the strings.

Recommended settings:
  - Material: PETG
  - Layers and perimeters
    - Perimeters: 7
    - Solid layers: Top: 6, Bottom: 6
  - Infill
    - Fill density: 40%
    - Fill pattern: Cubic
